

# إضافة البحث والفلترة لجدول البيانات

## التغييرات على `src/components/DataTableView.tsx`

### 1. شريط أدوات علوي (Toolbar) فوق الجدول
- **حقل بحث بالاسم**: Input للبحث النصي الفوري في أسماء الأفراد
- **فلتر الجنس**: Select (الكل / ذكر / أنثى)
- **فلتر "عرض ذرية جد"**: Select يعرض قائمة بالأجداد (الأشخاص الذين لديهم أبناء)، عند الاختيار يُعرض الجد وكامل ذريته بشكل متسلسل (أبناء، أحفاد، إلخ)
- **عداد النتائج**: يعرض عدد الأفراد المطابقين
- **زر مسح الفلاتر**: لإعادة عرض الكل

### 2. منطق الفلترة
- دالة `getDescendants(id)` تبني قائمة كاملة بكل أحفاد شخص معين بشكل تكراري (recursive)
- الفلاتر تعمل معاً: البحث بالاسم + الجنس + فرع الجد كلها تُطبق في آن واحد
- قائمة الأجداد تُبنى من `childrenMap` (كل شخص له أبناء يظهر في القائمة)

### 3. التصميم
- الشريط العلوي ثابت فوق الجدول مع خلفية `bg-card` وحدود سفلية
- اتجاه RTL متوافق
- متجاوب مع الشاشات الصغيرة (flex-wrap)
