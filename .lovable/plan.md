

# مراجعة شاملة وتنفيض كامل للمنصة

## المشاكل المكتشفة

### 1. صفحة الشجرة (الأهم — "بدائي وكئيب")
- **خلفية الشجرة شفافة** (`background: 'transparent'`) — تبدو فارغة وباهتة
- **أزرار الزوم** بسيطة جداً ومنفصلة عن الهوية البصرية
- **بطاقات الأشخاص** (220x90px): التصميم مسطح، بدون عمق بصري كافي، الشارات التراثية تزاحم المساحة
- **الخطوط الواصلة** بين البطاقات رفيعة وباهتة
- **PersonDetails** (الشريط الجانبي): تصميم عادي جداً بدون هوية تراثية

### 2. مشاكل تصميمية عامة عبر الصفحات
- **عدم تناسق** بين صفحات المنصة — كل صفحة لها طابع مختلف
- **ألوان الفروع** (`BRANCH_COLORS`) تبدو عشوائية ولا تنسجم مع هوية المنصة التراثية
- **الداكن موحش**: ألوان الوضع الداكن كئيبة (أزرق قاتم بدلاً من أخضر/رملي داكن دافئ)

### 3. تفاصيل صغيرة لكن مؤثرة
- خلفية `canvas-bg` لا تحتوي أي نقش أو تأثير بصري
- زر `ThemeToggle` و `FontSizeToggle` تصميمهم outline/ghost يبدو مختلف عن باقي العناصر
- `DataTableView` جدول عادي بدون لمسة تراثية

## خطة الإصلاح

### المرحلة 1: تحسين ألوان وأجواء المنصة (index.css)
- **Light mode**: إضافة CSS variable جديد `--canvas-pattern` لنقش خلفي خفيف
- **Dark mode**: تغيير الألوان من أزرق قاتم بارد إلى أخضر/بني داكن دافئ يتناسب مع الهوية التراثية
- إضافة CSS class `canvas-pattern` بتأثير خلفية radial-gradient خفيف كنقش رملي

### المرحلة 2: إعادة تصميم صفحة الشجرة (الأولوية القصوى)
**`FamilyTree.tsx`:**
- إضافة خلفية بنقش خفيف (radial dots pattern) بدلاً من الشفافية الفارغة
- تحسين أزرار الزوم: تصميم pill أنيق مدمج (عمودي مع فواصل)
- إضافة MiniMap خفيفة في الزاوية (اختياري)

**`FamilyCard.tsx`:**
- تحسين البطاقات: ظلال أعمق، حدود مزدوجة تراثية خفيفة
- خلفية بتأثير glassmorphism خفيف (backdrop-blur)
- تحسين الألوان الجندرية (أكثر دفئاً وأقل تشبعاً)
- تقليل الشارات على البطاقة الصغيرة (فقط المؤسس والمتوفي)

**`SpouseCard.tsx`:**
- تحسين التصميم ليكون أكثر أناقة ومطابقة للبطاقات الرئيسية

**`PersonDetails.tsx`:**
- إعادة تصميم الشريط الجانبي بهوية تراثية: خط ذهبي علوي، أيقونة أجمل، spacing أفضل
- إضافة زر "عرض النسب" يأخذك لصفحة الشخص

### المرحلة 3: توحيد التصميم عبر الصفحات

**`useTreeLayout.ts`:**
- ألوان فروع أكثر دفئاً وتناسقاً مع الهوية (بني، زيتي، كستنائي، بنفسجي دافئ)
- زيادة `strokeWidth` للخطوط من 2 إلى 2.5

**`AppHeader.tsx`:**
- تحسين الـ segmented control: حدود ذهبية خفيفة، active state أوضح
- تحسين الـ mobile bottom nav: أيقونة active أكبر مع نقطة ملونة

**`KinshipCalculator.tsx`:**
- تحسين بسيط: إضافة حد ذهبي علوي لبطاقة النتيجة

**`DataTableView.tsx`:**
- إضافة header تراثي مع عنوان وشارة
- تحسين ألوان الصفوف المتناوبة

**`ListView.tsx`:**
- تحسين بسيط: ألوان أدفأ للـ depth bar

### المرحلة 4: تحسين الوضع الداكن (index.css)
- تغيير `--background` من أزرق بارد `220 30% 8%` إلى أخضر/بني دافئ `160 20% 8%`
- تغيير `--card` لتتناسب
- تغيير `--muted` لدرجات أدفأ
- تحسين `--canvas-bg` للوضع الداكن
