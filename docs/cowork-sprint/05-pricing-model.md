### الدروس المستفادة:

1. **FamilySearch يهيمن على السوق المجاني** — استثمار حكومي ضخم (الكنيسة الأمريكية)
2. **الأسعار العالمية غالية على السعوديين** — ٩٩ دولار = ٣٧٠ ريال تقريباً
3. **السوق العربي يفتقر لتطبيقات احترافية** — معظمها بدائي أو غير محدث
4. **النماذج المخصصة تستهدف العوائل الثرية** فقط
5. **التسعير غير الواضح** نقطة ضعف مشتركة عند المنافسين المحليين

## ب) ثلاث سيناريوهات تسعير مقترحة

### السيناريو الأول: نموذج الباقات (Subscription Tiers)

**الفكرة**: ثلاث باقات شهرية/سنوية، مثل Ancestry.

#### الباقات:

| الميزة | **الأساسي** | **الفضي** | **الذهبي** |
|--------|-----------|---------|----------|
| **السعر/شهر** | ٩٩ ر | ١٩٩ ر | ٢٩٩ ر |
| **السعر/سنة** | ٨٩٩ ر | ١٧٩٩ ر | ٢٦٩٩ ر | |
| أعضاء العائلة | ٥٠ | ٢٠٠ | غير محدود |
| شجرة تفاعلية | ✓ | ✓ | ✓ |
| حاسبة القرابة | ✓ | ✓ | ✓ |
| لوحة إدارة | - | ✓ | ✓ |
| تقارير متقدمة | - | - | ✓ |
| طباعة شجرة PDF | - | ✓ | ✓ |
| دعم الصور | ١GB | ٥GB | ١٠GB |
| تحميل البيانات CSV | - | ✓ | ✓ |
| حساب الباحث | - | ✓ | ✓ |

#### حساب الإيرادات:

**السيناريو المحافظ**:
- ١٠ عوائل × ١٠ منها الأساسي (٨٩٩ر) + ٥ منها الفضي (١٧٩٩ر) =
- (١٠ × ٨٩٩) + (٥ × ١٧٩٩) = ٨,٩٩٠ + ٨,٩٩٥ = **١٧,٩٨٥ ريال سنوياً**

**السيناريو الواقعي**:
- ٣٠ عائلة × ١٠ منها الأساسي + ١٢ منها الفضي + ٨ منها الذهبي =
- (١٠ × ٨٩٩) + (١٢ × ١٧٩٩) + (٨ × ٢٦٩٩) = ٨,٩٩٠ + ٢١,٥٨٨ + ٢١,٥٩٢ = **٥٢,١٧٠ ريال سنوياً**

**السيناريو الإيجابي**:
- ١٠٠ عائلة × ٣٠ أساسي + ٤٠ فضي + ٣٠ ذهبي =
- (٣٠ × ٨٩٩) + (٤٠ × ١٧٩٩) + (٣٠ × ٢٦٩٩) = ٢٦,٩٧٠ + ٧١,٩٦٠ + ٨٠,٩٧٠ = **١٧٩,٩٠٠ ريال سنوياً**

---

### السيناريو الثاني: نموذج حسب الحجم (Per-User Pricing)

**الفكرة**: سعر ثابت لكل عضو عائلة + رسم قاعدة.

#### التسعير:

- **رسم الأساس**: ٢٩٩ ريال/سنة (للعائلة)
- **سعر العضو**: ٥ ريالات لكل عضو إضافي

#### أمثلة:
- عائلة من ٥٠ عضو = ٢٩٩ + (٥٠ × ٥) = ٥٤٩ ريال/سنة
- عائلة من ٢٠٠ عضو = ٢٩٩ + (٢٠٠ × ٥) = ١,٢٩٩ ريال/سنة
- عائلة من ٥٠٠ عضو = ٢٩٩ + (٥٠٠ × ٥) = ٢,٧٩٩ ريال/سنة

#### حساب الإيرادات:

**السيناريو المحافظ**:
- ١٠ عوائل بمتوسط ٢٠٠ عضو لكل عائلة =
- ١٠ × [٢٩٩ + (٢٠٠ × ٥)] = ١٠ × ١,٢٩٩ = **١٢,٩٩٠ ريال سنوياً**

**السيناريو الواقعي**:
- ٣٠ عائلة بمتوسط ١٥٠ عضو لكل عائلة =
- ٣٠ × [٢٩٩ + (١٥٠ × ٥)] = ٣٠ × ١,٠٤٩ = **٣١,٤٧٠ ريال سنوياً**

**السيناريو الإيجابي**:
- ١٠٠ عائلة بمتوسط ٣٠٠ عضو لكل عائلة =
- ١٠٠ × [٢٩٩ + (٣٠٠ × ٥)] = ١٠٠ × ١,٧٩٩ = **١٧٩,٩٠٠ ريال سنوياً**

---

### السيناريو الثالث: نموذج استشاري (Consultation Model)

**الفكرة**: تسعير مخصص حسب احتياجات كل عائلة، دون باقات معلنة.

#### الخيارات:

| الحزمة | السعر | ما يشمل | المدة |
|--------|------|--------|------|
| **الاستشارة الأولية** | ٥٠٠ ر | تقييم احتياجات، عرض توضيحي شخصي | جلسة واحدة |
| **الإعداد الأساسي** | ٢,٩٩٩ ر | بناء الشجرة، إدخال ٢٠٠ عضو | دعم شهر واحد |
| **الإعداد الكامل** | ٩,٩٩٩ ر | بناء كامل، إدخال ٥٠٠ عضو، تدريب | دعم ٣ أشهر |
| **الخطة الذهبية** | ١٩,٩٩٩ ر | كل شيء + تصميم شعار + كتاب عائلة PDF | دعم سنة واحدة |
| **الدعم المستمر** | ٩٩ ر/شهر | تحديثات، إضافة أعضاء جدد، الصيانة | متجدد |

#### حساب الإيرادات:

**السيناريو المحافظ**:
- ٥ عوائل استشارة (٥٠٠ × ٥) + ٣ عوائل إعداد أساسي (٢,٩٩٩ × ٣) + دعم مستمر ٢ عائلة (٩٩ × ٢ × ١٢) =
- ٢,٥٠٠ + ٨,٩٩٧ + ٢,٣٧٦ = **١٣,٨٧٣ ريال سنوياً** (بدون تكرار الاستشارات)

**السيناريو الواقعي**:
- ٢ إعداد كامل (٩,٩٩٩ × ٢) + ٥ خطط ذهبية (١٩,٩٩٩ × ٥) + ١٠ عوائل دعم مستمر (٩٩ × ١٠ × ١٢) =
- ١٩,٩٩٨ + ٩٩,٩٩٥ + ١١,٨٨٠ = **١٣١,٨٧٣ ريال سنوياً**

**السيناريو الإيجابي**:
- ١٠ عوائل إعداد كامل (٩,٩٩٩ × ١٠) + ١٥ خطة ذهبية (١٩,٩٩٩ × ١٥) + ٥٠ عائلة دعم (٩٩ × ٥٠ × ١٢) =
- ٩٩,٩٩٠ + ٢٩٩,٩٨٥ + ٥٩,٤٠٠ = **٤٥٩,٣٧٥ ريال سنوياً**

## هـ) السجل التجاري — متى وكيف؟

### متى تحتاج إلى سجل تجاري؟

**الإجابة**: عندما تبدأ استقبال الدفعات الفعلية من عوائل غير الخنيني.

**الحد الأدنى القانوني السعودي**:
- ❌ **غير مطلوب** إذا كانت إيراداتك أقل من ~٥٠ ألف ريال سنوياً
- ✓ **مطلوب** إذا تجاوزت ذلك أو إذا أردت القانونية الكاملة

**للأمان القانوني والتسويق**:
- يفضل التسجيل قبل وصول ٥-١٠ عوائل تدفع

### الخطوات لفتح سجل تجاري في السعودية:

#### الخطوة ١: تجهيز المستندات (١ يوم)

- ✓ بطاقة الهوية الوطنية أو جواز السفر
- ✓ شهادة حسن السيرة (من مركز الشرطة)
- ✓ عنوان المقر (منزلك يكفي، أو عنوان تجاري)
- ✓ وصف النشاط: "خدمات تكنولوجيا المعلومات / تطبيقات شجرة العائلة"
- ✓ رقم الهوية البريدية الوطنية (تطبيق Address)

#### الخطوة ٢: فتح الحساب البنكي (١-٣ أيام)

- اختر بنكاً سعودياً (الأهلي، الراجحي، إلخ)
- اشرح للبنك أنك ستستقبل دفعات من العملاء
- افتح حساب تجاري (يفضل)

#### الخطوة ٣: التسجيل في المركز السعودي للأعمال (١ يوم)

- **الموقع**: [business.sa](https://business.sa)
- **الخطوات**:
  1. أنشئ حساب
  2. اختر "سجل تجاري جديد"
  3. أدخل بيانات النشاط
  4. أدخل رأس المال المتوقع (ادفع ٢٠,٠٠٠ ريال كتصريح، أو ابدأ بـ ٥,٠٠٠)
  5. ادفع الرسوم
  6. احصل على السجل فوراً (ورقياً خلال يوم)

#### الخطوة ٤: التسجيل في الغرفة التجارية (٢-٣ أيام)

- ستوجه تلقائياً بعد السجل التجاري
- أدفع رسوم الغرفة (تختلف حسب النشاط، عادة ١٠٠-٥٠٠ ريال)

#### الخطوة ٥: التسجيل في الزكاة والضريبة (اختياري/إجباري حسب الإيرادات)

- إذا تجاوزت ١ مليون ريال سنوياً، يصبح إجبارياً
- **الموقع**: [zatca.gov.sa](https://zatca.gov.sa)

| السجل التجاري الرئيسي | ٢٠٠ ر | من وزارة التجارة |
| رسم الغرفة التجارية | ٢٠٠-٥٠٠ ر | حسب النشاط والموقع |
| **الإجمالي** | **٤٠٠-٧٠٠ ريال** | يمكن إنهاؤه في أسبوع |

**إضافات اختيارية**:
- رسم الترخيص البلدي (١,٠٠٠-٢,٠٠٠ ر) — إذا كان لديك مقر فعلي
- استشاري قانوني (١,٠٠٠-٢,٠٠٠ ر) — يساعدك في المتاهة

### الخطة الموصى بها (لـ Abdullah):

**المرحلة الأولى (الآن)**: عمل غير رسمي، استقبل الدفعات كفرد
- إذا استخدمت حسابك الشخصي مع تحويلات بنكية، الجميع يفهمون

**المرحلة الثانية (٣-٦ أشهر)**: عندما تصل إلى ٥-١٠ عوائل تدفع
- سجل رسمي (~٥٠٠ ريال)
- حساب بنكي تجاري

**المرحلة الثالثة (١-٢ سنة)**: عندما تصل إلى ٥٠+ عائلة
- شركة ذات مسؤولية محدودة (أغلى، ~٢,٠٠٠ ريال + رسوم قانونية)
- فاتورة إلكترونية (إلزامي منذ ٢٠٢٤)
