### ٤. نقطة التعادل (Break-Even)

```
معادلة البداية:
الإيراد المتوقع = (عدد العوائل المدفوعة) × (السعر المتوسط) - (رسوم المعالجة)
التكاليف = التكاليف الثابتة + (عدد العوائل × التكاليف المتغيرة)

Break-Even عندما:
الإيراد = التكاليف
```

**الحساب**:
```
السيناريو الأساسي (Average)
- السعر المتوسط: ٦٠٠ ريال
- رسوم المعالجة: ٣% = ١٨ ريال
- الإيراد الفعلي لكل عائلة: ٥٨٢ ريال

- التكاليف الثابتة: ٤ ريال/شهر
- التكاليف المتغيرة: ٣٥٠ ريال لكل عائلة

Break-Even:
582X = 4 + 350X
232X = 4
X = 0.017 ≈ 1 عائلة لكل شهر = break-even تقريباً

الخلاصة: أول عائلة مدفوعة = break-even فوري!
```

### ٥. توقعات الإيرادات (السيناريوهات)

#### السيناريو الدعوبي (Pessimistic)
```
الفرضيات:
- ٠ عوائل مدفوعة بعد ٩٠ يوم
- ٥٠ مسجّل فقط من الخنيني
- معدل تحويل: ٠%

الإيرادات:
- ٣ شهر (مارس-يونيو): ٠ ريال
- ٦ شهر (مارس-أغسطس): ٠ ريال
- ١٢ شهر (مارس ٢٠٢٦ - مارس ٢٠٢٧): ~٥٠٠ ريال (عائلة واحدة متأخرة)
```

#### السيناريو المتوسط (Base Case)
```
الفرضيات:
- ٤ عوائل مدفوعة بعد ٩٠ يوم
- ١٥٠ مسجّل (الخنيني + عوائل أخرى)
- معدل تحويل: ٢٫٧%
- السعر المتوسط: ٦٠٠ ريال

الإيرادات:
- ٣ شهر: ١,٦٠٠ ريال (٤ عوائل × ٤٠٠ ريال متوسط بعد رسوم)
- ٦ شهر: ~٥,٠٠٠ ريال (بناءً على النمو التدريجي)
- ١٢ شهر: ~١٥,٠٠٠-٢٠,٠٠٠ ريال
```

#### السيناريو المتفائل (Optimistic)
```
الفرضيات:
- ١٠ عوائل مدفوعة بعد ٩٠ يوم (بما فيها الخنيني)
- ٣٠٠ مسجّل
- معدل تحويل: ٣٫٣%
- السعر المتوسط: ٧٠٠ ريال (مزيج من ٥٠٠-١٠٠٠)

الإيرادات:
- ٣ شهر: ٥,٠٠٠ ريال
- ٦ شهر: ~١٥,٠٠٠ ريال
- ١٢ شهر: ~٥٠,٠٠٠ ريال (شهري متوسط ٤,١٦٧ ريال)
```

|--------|----------|------------|----------|--------|----------|---------|----------|
