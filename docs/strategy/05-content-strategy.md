### ١.٢ قسم المميزات (Features Section)

#### المميزات الحالية
- شجرة تفاعلية — تصفح شجرة عائلتك بصرياً واكتشف صلات القرابة بين أي فردين
- خاصة وآمنة — منصتك الخاصة برابط مخصص، بياناتك لعائلتك فقط
- تعمل على الجوال — تطبيق ويب متكامل يعمل على كل الأجهزة بدون تحميل

#### النصوص المحسّنة

**١) شجرة تفاعلية**
- النص الحالي: تصفح شجرة عائلتك بصرياً واكتشف صلات القرابة بين أي فردين
- النص المحسّن:
  ```
  شجرة تفاعلية ملونة توضح العلاقات بين أفراد العائلة — زوم، تصفح،
  ابحث عن أي عضو، واكتشف القرابة بينك وبين أي فرد في الشجرة بضغطة زر
  ```

**٢) خاصة وآمنة**
- النص الحالي: منصتك الخاصة برابط مخصص، بياناتك لعائلتك فقط
- النص المحسّن:
  ```
  منصة مغلقة بطابع عائلتك — رابط خاص وكلمة مرور وأرقام سرية،
  بياناتك تحت سيطرتك والأصدقاء فقط من تختارهم يشوفوا البيانات
  ```

**٣) تعمل على الجوال**
- النص الحالي: تطبيق ويب متكامل يعمل على كل الأجهزة بدون تحميل
- النص المحسّن:
  ```
  متاح على كل الأجهزة بدون تحميل تطبيق — على الجوال أو الحاسب أو الجدول،
  وتشتغل حتى بدون إنترنت (تقنية PWA)
  ```

## ٨. نبرة الصوت (Tone of Voice)

### ٨.١ القيم والمبادئ

**نسبي يتكلم مع جمهوره بـ:**

#### 1️⃣ التودد والألفة
```
❌ لا تقول: "تثبيت منصة العائلة متطلب أساسي"
✅ قول: "خلّي عائلتك توثق نفسها بسهولة"

❌ لا تقول: "بيانات المستخدم محمية بتشفير AES-256"
✅ قول: "بياناتك آمنة مثل الحساب البنكي"
```

#### 2️⃣ الوضوح والبساطة
```
❌ لا تقول: "نموذج العلاقات الهرمية يُعرّف الأسلاف والنسل"
✅ قول: "ابحث عن أي شخص واعرف أبوه وأبناؤه"

❌ لا تقول: "واجهة المستخدم تتميز بـ UX محسّن"
✅ قول: "سهل الاستخدام حتى للجدة"
```

#### 3️⃣ الاحترام الثقافي
```
❌ لا تقول: "Family tree"
✅ قول: "شجرة العائلة" أو "النسب"

✅ استخدم كلمات عائلية: "الإخوة"، "الأجداد"، "الإرث"، "التراث"
```

#### 4️⃣ الإيجابية والتشجيع
```
✅ "ابدأ الآن — أنشئ منصة عائلتك"
✅ "وثّق تراثك قبل فوات الأوان"
✅ "كل عضو يحدث بيانات نفسه — سهل وآمن"
```

#### 5️⃣ الصراحة والثقة
```
✅ "لا توجد رسوم مستترة — مجاني تماماً الآن"
✅ "في التحديثات القادمة سنضيف الصور"
✅ "لو عندك مشكلة، تواصل معنا — سنساعدك"
```

---

### ٨.٢ الأسلوب حسب السياق

#### أ) المرحلة البطولية (Hero Copy)
```
نبرة: ملهمة + عاطفية + قوية

مثال:
"وثّق تراث عائلتك واحفظه للأجيال القادمة"
"كل اسم له قصة — احفظها رقمياً"
"شجرتك تستحق منصة خاصة"
```

#### ب) أزرار الـ CTA
```
نبرة: فعل + حماس + وضوح

أمثلة:
"ابدأ الآن — أنشئ منصة عائلتك"
"جرب الديمو مجاناً"
"فعّل منصة عائلتك اليوم"
"شوف كيف تشتغل"

❌ تجنب: "Submit"، "Continue"، "Next"
```

#### ج) الدعم والمساعدة
```
نبرة: ودية + داعمة + سريعة

مثال:
"أول مرة معك؟ سنساعدك خطوة بخطوة"
"عندك سؤال؟ تواصل معنا — سنرد في دقائق"
"سنجعلها سهلة لك 😊"
```

#### د) الأخطاء والمشاكل
```
نبرة: معتذرة + حلول + إيجابية

مثال:
"عذراً، حصل خطأ. لكن سنحله لك الآن"
"هذا الحقل مطلوب — ركز على هذا"
"الرقم يجب أن يكون بهذا الشكل: 05xxxxxxxx"

❌ تجنب: "Error 404"، "Invalid Input"
```

#### ه) الإحصائيات والنجاحات
```
نبرة: فخر + تحفيز + واقعية

أمثلة:
"عائلة الخنيني الآن فيها +٤١٠ عضو موثق"
"اختبرها ملايين الأسر السعودية"
"أول منصة عربية متخصصة في النسب"
```

#### و) الضمانات والأمان
```
نبرة: واثقة + قطعية + آمنة

أمثلة:
"بيانات عائلتك محمية مثل البنك"
"لا نشارك بيانات أحد — أبداً"
"أنت تتحكم كاملاً من يدخل"

❌ تجنب: "Might be safe"، "Probably secure"
```

### ٨.٣ قائمة الكلمات المفضلة والممنوعة

#### الكلمات المفضلة:
```
✅ استخدم:
- وثّق / توثيق
- تراث / إرث
- أمان / آمنة
- خاص / خصوصية
- شجرة / فرع
- نسب / قرابة
- عائلة / أسرة
- ابدأ / ابدي
- سهل / بسيط
- جاهز / مجهز
```

#### الكلمات الممنوعة:
```
❌ تجنب:
- "بيانات": استخدم "معلومات"
- "خادم": استخدم "منصة"
- "تطبيق": استخدم "منصة" (إلا إذا تحدثت عن الموبايل)
- "عضو": استخدم "فرد" أحياناً للتنويع
- "تحديث": استخدم "تحديث" أو "تطوير"
- كلمات إنجليزية غير ضرورية
```

### ٨.٤ أمثلة عملية لنبرة الصوت

#### مثال ١: رسالة خطأ

❌ **سيء:**
```
Error: Invalid phone number format. Expected: +966XXXXXXXXX
```

✅ **جيد:**
```
رقم الجوال غير صحيح.

الشكل الصحيح: 05xxxxxxxx
مثال: 0555551234
```

#### مثال ٢: طلب تسجيل

❌ **سيء:**
```
Please enter your Hijri birth date (optional)
```

✅ **جيد:**
```
تاريخ ميلادك بالهجري (اختياري)

يساعد في التمييز بينك وبين من تشاركه الاسم
```

#### مثال ٣: رسالة ترحيب

❌ **سيء:**
```
Welcome to the platform. You have been successfully registered.
```

✅ **جيد:**
```
أهلاً وسهلاً بك! 🎉

أنت الآن موثق في شجرة عائلتك.
ادخل المنصة واستكشف أسرتك الآن.
```

#### مثال ٤: نص الميزة

❌ **سيء:**
```
Advanced hierarchical relationship computation engine
```

✅ **جيد:**
```
⚡ حاسبة القرابة الذكية

اختر أي شخصين واعرف قرابتهما فوراً:
- هذا عمي؟
- هذا ابن خالي؟
- هذا قريب؟
```

#### مثال ٥: رسالة الدعم

❌ **سيء:**
```
Contact our support team via email at support@nasaby.app for assistance.
```

✅ **جيد:**
```
محتاج مساعدة؟ نحن هنا! 👋

📞 واتساب: [رقم]
📧 بريد: support@nasaby.app

سنرد عليك في أقرب وقت 🚀
```

### ٨.٥ معايير الجودة

عند كتابة أي محتوى لنسبي، تأكد من:

```
☑ عربية صحيحة (بدون أخطاء إملائية)
☑ ألفاظ عائلية واحترام ثقافي
☑ بساطة ووضوح (حتى للجدة)
☑ إيجابية وتشجيع
☑ عدم الإطالة (جمل قصيرة)
☑ استخدام الإمojis بذكاء (بدون إفراط)
☑ نداء عاطفي (ليس فقط معلومات)
☑ CTA واضحة وقوية
☑ توافق مع قيمنا الأسرية
☑ سهولة القراءة والفهم
```
