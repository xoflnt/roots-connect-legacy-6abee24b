# خريطة طريق منصة نسبي (Nasaby)
## الإصدار ١.٠ — استراتيجية النمو من عائلة إلى ١٠٠ عائلة

**آخر تحديث:** ٢٨ مارس ٢٠٢٦
**المسؤول:** عبدالله (مؤسس / مطور بمساعدة Claude Code)
**الحالة:** في المسار — عائلة الخنيني (٤١٠+ عضو) — جاهزة للتوسع

## 🎯 الرؤية

تصبح نسبي **منصة SaaS العربية الأولى لشجرات العوائل** — توفر لكل عائلة سعودية/خليجية:
- شجرة تفاعلية جميلة لاكتشاف الأنساب
- أداة قرابة دقيقة (قرابة شرعية)
- إدارة بيانات آمنة وخاصة
- تجربة موبايل أولاً (بدون تطبيق — ويب فقط)
- سهولة الشراء والتفعيل (لا تعقيد تقني)

**المقاييس الناجحة:**
- ٢٠٠ عائلة مسجلة (نهاية ٢٠٢٦)
- ٥ نجوم في AppStore / Play Store (ويب تطبيق)
- ٩٥% رضا العملاء
- ٥ عائلات تدفع اشتراك شهري/سنوي

## 📊 جرد المميزات الحالية (الخنيني v1.0)

### ✅ المميزات الأساسية المكتملة

#### 1. **الشجرة التفاعلية** (React Flow)
- تصفح بصري كامل للعائلة مع ١٠+ أجيال
- عرض الأب-الأبناء مع ترميز ألوان حسب الفروع (٣ فروع)
- إمكانية التوسيع والطي (Expand/Collapse) بنقرة
- خاصية "ابحث عني" لتحديد موقعك بالشجرة
- عرض الأجيال بشرائط لونية (Generation Bands)
- مصغر الخريطة (MiniMap) للتنقل السريع
- تصفية حسب: الفرع، الجنس، الأحياء/الأموات
- زوم كامل مع تحريك حر (Dragging)

#### 2. **حاسبة القرابة** (Kinship Calculator)
- حساب درجة القرابة بين أي شخصين (LCA-based)
- ترجمة دقيقة لـ ١٧٠+ علاقة قرابة عربية
- كشف الإخوة الأشقاء من الإخوة لأب (بناءً على اسم الأم)
- عرض مسارات الصعود (Ancestor Chains)
- ثلاث طرق عرض: شجرة / بطاقات / وثيقة قانونية
- دعم الاتجاهين: "أنت عم له" ↔ "هو ابن أخيك"

#### 3. **المصادقة المخصصة** (Custom Auth)
- لا توجد حسابات Gmail/Supabase Auth
- مصادقة برمز عائلي (6 أرقام) + رقم هاتف سعودي
- تحديث اختياري لسنة الميلاد الهجرية
- حفظ في `localStorage` بدون انقضاء
- صفحة الملف الشخصي مع خيارات التحديث الذاتي

#### 4. **لوحة إدارة كاملة** (Admin Dashboard)
- عرض الإحصائيات: عدد الأعضاء، الأجيال، التوزيع حسب الفروع
- إدارة الأعضاء: إضافة / تحرير / حذف / أرشفة
- مراجعة الطلبات (إضافة مولود، زواج، ملاحظات)
- إرسال إشعارات broadcast للعائلة
- عرض المستخدمين المسجلين (من دخلوا المنصة)
- فحص صحة البيانات (Data Health) — مراقبة الأيتام والمشاكل

#### 5. **نظام الطلبات** (Family Requests)
- تقديم طلب لإضافة مولود جديد (من أب معروف)
- تقديم طلب إضافة زواج
- ملاحظات حرة (other)
- تحديث الحالة: معلق → موافق ← مكتمل
- عرض الطلبات في ملفك الشخصي

#### 6. **الإشعارات** (Notifications)
- Web Push Notifications (RFC 8291)
- إشعارات الموافقة على الطلبات
- إشعارات الإذاعة (Broadcast)
- إشعارات الأعضاء الجدد
- قائمة الإشعارات مع "اقرأ" و"لم تقرأ"

#### 7. **صفحة العروض التوضيحية** (Demo Page)
- عرض ديموي كامل لعائلة عشوائية
- إدراج الزائر تلقائياً كعضو في الشجرة
- لا يتطلب مصادقة حقيقية
- بيان ديموي علوي + زر التواصل

#### 8. **صفحة الهبوط الرئيسية** (Landing Page)
- بطل جميل (Hero) مع صورة خلفية
- ثلاث بطاقات مميزات رئيسية
- عرض الإحصائيات: العدد الإجمالي، الأجيال، الذكور/الإناث
- بطاقات الفروع الثلاثة
- شريط سريع للبحث عن نفسك
- رابط التسجيل/الدخول

#### 9. **الهوية البصرية**
- تصميم RTL كامل (لغة عربية طبيعية)
- نمط ألوان التراث: أخضر وذهبي وبرتقالي
- خط YearOfHandicrafts (خط عربي متقن)
- وضع مظلم/فاتح (Dark/Light Mode)
- تطبيق Tatweel (الكشيدة) في الرؤوس
- أيقونات مخصصة (Badge Heritage)

#### 10. **التكنولوجيا الأساسية**
- React + Vite (بدون Next.js)
- Supabase (PostgreSQL + Edge Functions)
- React Flow (شجرة تفاعلية)
- d3-hierarchy (حساب التخطيط)
- PWA كامل مع Service Worker
- Offline Support
- Push Notifications

#### 11. **الأمان والخصوصية**
- Row Level Security (RLS) على Supabase
- جميع الكتابات عبر Edge Functions
- رموز إدارة الجلسة (٢ ساعة)
- قواعد خصوصية: الضيوف لا يرون الأعمار/الزوجات
- تشفير كامل للبيانات في قاعدة البيانات

#### 12. **البيانات الثابتة والديناميكية**
- Static data في `familyData.ts` (٤١٠+ عضو للخنيني)
- قاعدة بيانات Supabase للبيانات الإضافية (الأرشيف، التحديثات)
- استراتيجية دمج ذكية (Static = الأساس، Cloud = الإثراء فقط)
- بدون حذف الأساس أبداً

#### 2. **تسجيل عائلة جديدة** (Family Registration) — 🟠 عالي

**المطلوب:**
- نموذج التسجيل: اسم العائلة ← slug منخفض ← كلمة مرور إدارية ← رمز عائلي

- إنشاء بيانات أولية أساسية (مؤسس + 3 أبناء)
- إرسال رسالة ترحيب بـ slug
- إعادة توجيه إلى `/family/{slug}/admin`

**الجهد:** ١٥ ساعة | **الصعوبة:** متوسط | **التأثير:** ٩٥%

## 💎 مميزات "الـ WOW Factor" — اللي تخلي العائلة تشتري

### 1️⃣ **الشهادة الذهبية** (The Golden Certificate)
- شهادة نسب رسمية بتصميم فاخر (بخط عربي تقليدي)
- QR code يرابط للنسب على الويب
- توقيع إدارة العائلة الرقمي
- قابلة للطباعة على ورق عالي الجودة
- **السعر المضاف:** ٪٪٪ premium → عائلة تدفع اضافي لطباعة

### 2️⃣ **آلة الاكتشاف** (Discovery Engine)
- "هل أنت قريب لفلان؟" — بحث ذكي في كل العائلات المسجلة
- عرض درجة القرابة تلقائياً
- "ربما تعرف هؤلاء الأشخاص" — اقتراحات بناءً على النسب
- **التأثير على المبيعات:** FOMO — "تريد أن ترى من في العائلة معك؟"

### 3️⃣ **تقرير النسب السنوي** (Annual Pedigree Report)
- تقرير شامل: كم ولد، كم توفي، أطول نسب، أشهر اسم
- مقارنة مع السنة الماضية (نمو العائلة)
- إشعار تلقائي: "عائلتك نمت بـ ١٢ عضو هذا العام"
- **التأثير:** فخر بالعائلة + مشاركة على وسائل التواصل

### 4️⃣ **جدول الصلات** (Kinship Matrix)
- جدول ضخم يعرض درجة القرابة بين ٢٠ شخص مختار
- تصدير كـ Excel / PDF
- مفيد للتخطيط الاجتماعي والزواج (تشاور الأهل)
- **التأثير:** استخدام فريد لعائلة سعودية

### 5️⃣ **الشارة الموروثة** (Heritage Badge)
- كل عضو يحصل على شارة حسب موقعه: (مؤسس، جيل ١، جيل ٢، إلخ)
- شارات قابلة للمشاركة على السوشيال
- "أنا جيل ٣ من عائلة الخنيني منذ ١٣٠ سنة" 🏛️
- **التأثير:** فخر شخصي + تسويق فموي

### 6️⃣ **النسب الحي** (Live Pedigree)
- عرض نسب شخص واحد كـ شجرة رأسية (من المؤسس إلى نفسه)
- كل اسم قابل للنقر لمعرفة المزيد
- تصدير كـ PDF عمودي (يطبع بجودة عالية)
- **التأثير:** هدية عائلية ثمينة
